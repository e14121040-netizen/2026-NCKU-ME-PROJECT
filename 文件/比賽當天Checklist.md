# 比賽當天 Checklist

> 比賽前一天晚上 + 當天早上核對這份清單，確保萬無一失。

---

## 📦 前一天晚上

### 硬體確認
- [ ] 取物機器人完整組裝，6 足轉動順暢
- [ ] 手臂伺服歸位正常（指令 `h`）
- [ ] 運輸機器人 EV3 組裝完整
- [ ] 放置機構正常運作

### 電源
- [ ] 11.1V 鋰電池**充滿電**（電壓 ≥ 12.0V）
- [ ] EV3 電池**充滿電**（或備用 AA 電池 ×6）
- [ ] 備用電池已充滿

### 軟體
- [ ] Arduino 程式**最新版已上傳**
- [ ] EV3 程式**最新版已部署**
- [ ] 手機 App（`.apk`）已安裝且可正常啟動
- [ ] HC-05 已與比賽用手機配對

### 工具箱
- [ ] 十字螺絲起子（小）
- [ ] 六角扳手（M3）
- [ ] 尖嘴鉗
- [ ] 電氣膠帶（黑）
- [ ] 三用電表（量電壓用）
- [ ] 束線帶（備用）
- [ ] 杜邦線（備用，公-母、公-公各數條）

### 備品
- [ ] 備用 M3 螺絲 + 螺帽 + 墊圈
- [ ] 備用伺服馬達（SG90 ×1）
- [ ] 備用杜邦線
- [ ] 備用電池

---

## 🏁 比賽當天早上

### 到場後立即做
- [ ] 取出機器人，目視檢查有無鬆動
- [ ] 接上電池，確認 Arduino LED 常亮（電壓正常）
- [ ] 發送 `v` 指令確認電池電壓 ≥ 10.5V
- [ ] EV3 開機，確認螢幕顯示電壓正常

### 場地測試（如有時間）
- [ ] 取物機器人：藍芽連線 → 前進/後退/轉向 → 手臂歸位
- [ ] EV3：在場地黑線上跑一次循跡
- [ ] EV3：確認黃色/紅色停止線偵測正常

### 比賽前 5 分鐘
- [ ] 兩台機器人擺到起始位置
- [ ] 手機 App 連線 HC-05
- [ ] EV3 程式啟動，等待觸碰感測器
- [ ] 操作員確認熟悉操作流程
- [ ] 清空場地上的障礙物

---

## 🔄 比賽中注意事項

1. **操作員專注遙控**，不要分心看 EV3
2. 夾取零件前先對準，**不要急**
3. 放置零件到 EV3 上時確認**穩固不會掉**
4. 如果取物機器人失控 → **立即放開手機按鈕**（會發送停止 `0`）
5. 如果 EV3 脱軌 → **按 EV3 上方按鈕停止**，手動放回軌道重啟

---

## 🆘 緊急處理

| 狀況 | 處理方式 |
|------|---------|
| 藍芽斷線 | App 重新連線，如果連不上就重啟 Arduino |
| 機器人翻倒 | 扶正，檢查有無鬆脫，重新啟動 |
| 電池耗盡 | 換備用電池 |
| EV3 當機 | 長按背面按鈕強制關機，重開機重啟程式 |
| 螺絲鬆脫 | 用工具箱的螺絲起子鎖緊 |
