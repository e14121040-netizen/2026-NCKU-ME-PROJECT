# 故障排除指南

> 比賽或測試時遇到問題？找這份文件就對了。

---

## 🤖 取物機器人（八足 Theo Jansen）

### 藍芽相關

| 症狀 | 可能原因 | 解決方法 |
|------|---------|---------|
| 手機找不到 HC-05 | 未配對 | 手機設定 → 藍芽 → 搜尋 → 配對 HC-05（密碼 `1234`） |
| App 連線後馬達不動 | 鮑率不符 | 確認 HC-05 鮑率 = **38400**，Arduino `Serial1.begin(38400)` |
| 連線後偶爾斷線 | 電池電壓低 / 距離太遠 | 充電、靠近（< 10m）、檢查 HC-05 天線 |
| iOS 無法使用 | HC-05 不支援 BLE | HC-05 只支援 Android，iOS 需改用 HM-10 模組 |

### 馬達相關

| 症狀 | 可能原因 | 解決方法 |
|------|---------|---------|
| 馬達完全不轉 | 接線錯誤 / ENA 跳線帽沒拔 | 檢查接線圖、**拔掉 L298N 上 ENA/ENB 跳線帽** |
| 馬達方向反了 | IN1/IN2 接反 | 交換 IN1 和 IN2 的接線，或修改程式碼 |
| 前進時機器人歪斜 | 左右馬達速度不同 | 調整 `SPEED_FULL` 或在程式中加左右補償 |
| 伺服馬達抖動 | 供電不足 | PCA9685 的 V+ 需外接 5~6V 電源，不能只靠 Arduino 供電 |
| 伺服回應但角度不對 | 歸位值需校正 | 調整 `SHOULDER_HOME`、`ELBOW_HOME` 等常數 |

### 機構相關

| 症狀 | 可能原因 | 解決方法 |
|------|---------|---------|
| 走路時卡死 | 曲柄相位錯誤 / 螺絲太緊 | 確認同側每足相位差 90°、左右錯開 180°、鬆開螺絲加尼龍墊圈 |
| 足端軌跡異常 | 連桿比例不對 | 參考 `TheoJansen連桿設計參考.md` 重新檢查連桿尺寸 |
| 機器人翻倒 | 手臂太重 / 重心偏移 | 加配重或降低手臂安裝位置 |
| 夾爪夾不住零件 | 扭力不足 / 角度不夠 | 調整 `CLAW_OPEN/CLAW_CLOSE` 值或換更大扭力伺服 |

### 電源相關

| 症狀 | 可能原因 | 解決方法 |
|------|---------|---------|
| Arduino 不開機 | 電壓不夠 | 確認 LM2596 輸出 7~9V，用三用電表量測 |
| LED 快速閃爍 | 電池電壓過低（< 9.6V） | **立即停止使用，充電！** |
| LED 慢速閃爍 | 電池電壓偏低（< 10.0V） | 可繼續使用但應盡快充電 |
| 突然斷電 | 電池保護板斷電 | 電池電壓過低觸發保護，充電後恢復 |

## 🚚 運輸機器人（EV3）

### 循跡相關

| 症狀 | 可能原因 | 解決方法 |
|------|---------|---------|
| 車子完全不跟線 | 感測器高度太高 | 調整離地 5~10mm |
| 嚴重左右晃動 | KP 太大 | 降低 `KP`，先從 0.5 開始調 |
| 晃動不收斂（越晃越大） | KD 不夠 | 增加 `KD` 值 |
| 慢慢偏離 | KI 不夠 | 微調增加 `KI`，但不要太大 |
| 速度太快失控 | BASE_SPEED 太高 | 降低 `BASE_SPEED`，穩定後再加 |

### 顏色偵測相關

| 症狀 | 可能原因 | 解決方法 |
|------|---------|---------|
| 黃色/紅色偵測不到 | 感測器離地太高 / 光線干擾 | 降低感測器高度、避免強光直射 |
| 錯誤觸發停止 | 地面反光 / 顏色相近 | 使用更鮮豔的色紙、加深顏色對比 |
| 偵測到但沒停車 | 程式邏輯問題 | 在 `check_stop_line()` 加 `print` 除錯 |

### EV3 連線相關

| 症狀 | 可能原因 | 解決方法 |
|------|---------|---------|
| EV3 開不了機 | 電池沒電 | 充電或更換 AA 電池 |
| 電腦連不到 EV3 | microSD 卡問題 | 確認已安裝 Pybricks OS、USB 線正常 |
| 程式上傳失敗 | VS Code 擴充問題 | 重新安裝 EV3 MicroPython 擴充 |
| 螢幕顯示 "ERROR!" | 程式執行異常 | 查看螢幕上的錯誤訊息，對照程式碼除錯 |
| 螢幕顯示 "Timeout!" | 循跡逾時 | 檢查軌跡線是否完整、感測器是否偏移 |

## ⚡ 快速排查流程

遇到問題時，按照以下順序快速排查：

```
1. 電源正常嗎？
   └─ 沒電 → 充電 / 換電池

2. 接線正確嗎？
   └─ 鬆了 → 重新插好、對照接線圖

3. 程式有上傳成功嗎？
   └─ 沒有 → 重新上傳

4. Serial Monitor 有輸出嗎？
   └─ 有 → 看 CMD 是否正確收到指令
   └─ 沒有 → 檢查 Serial 鮑率

5. 單獨測試子系統
   └─ 馬達、藍芽、伺服分開測
```
