# Chebyshev Lambda Linkage 六足設計參考

## 連桿機構原理

Chebyshev Lambda Linkage 是一個四桿連桿機構，能將曲柄的旋轉運動
轉換為足端的「近似直線 + 弧線」軌跡，形成自然的步行運動。

```
         固定支點 O1          固定支點 O2
              ●───────────────●
             /  機架 (Ground)   \
      曲柄  /                    \  搖桿
     (Crank)/                     \ (Rocker)
           /                       \
          ● A                     ● B
           \                     /
            \    耦桿 (Coupler) /
             \                 /
              ●───────────────●
                   足端 P
                 (Foot Point)
```

足端 P 的軌跡：下半段近似直線（接地行走），上半段弧線（抬腳擺動）。

## 推薦連桿尺寸

Chebyshev 經典比例（近似直線效果最佳）：

| 桿件 | 比例 | 建議尺寸 (mm) |
|------|------|---------------|
| 曲柄 (Crank) | 1 | 25 ~ 35 |
| 連桿 (Coupler) | 2.5 | 62.5 ~ 87.5 |
| 機架 (Ground) | 2.5 | 62.5 ~ 87.5 |
| 搖桿 (Rocker) | 2.5 | 62.5 ~ 87.5 |

### 建議起始參數（先用這組測試）
- **曲柄** = 30 mm
- **連桿** = 75 mm
- **機架** = 75 mm
- **搖桿** = 75 mm

此尺寸產生約 **~60mm 步幅高度**、**~80mm 步距**。

## 六足配置

```
  俯視圖（機身長軸方向 ↑ = 前進方向）

        前
    足1 ─── 足4
     |       |
    足2 ─── 足5       ← 機身
     |       |
    足3 ─── 足6
        後

  左側（足1,2,3）→ 馬達 A 驅動
  右側（足4,5,6）→ 馬達 B 驅動
```

### 三角步態 (Tripod Gait)

```
  Phase 1：足1, 足5, 足3 著地（三角△支撐）
           足4, 足2, 足6 擺動

  Phase 2：足4, 足2, 足6 著地（三角△支撐）
           足1, 足5, 足3 擺動
```

左右兩側連桿的曲柄錯開 180°，自然形成三角步態。

### 同側足的相位差

同一側 3 足的曲柄相位：
- 足1 和 足3：同相（0°）
- 足2：反相（180°）

這樣任何時刻同側都有 2 足著地、1 足擺動。

## 材料選擇

| 材料 | 優點 | 缺點 | 適用 |
|------|------|------|------|
| **壓克力 (3mm)** | 便宜、雷切精準、透明美觀 | 易碎（跌落可能斷裂） | 連桿片 |
| **3D 列印 PLA** | 設計自由度高、可做複雜形狀 | 精度略低、層間強度差 | 機身、支架 |
| **鋁板 (2mm)** | 強度高、耐用 | 加工較難、成本高 | 關鍵承力件 |

### 推薦組合
- 連桿片：**壓克力雷切**（便宜快速）
- 機身底盤：**壓克力 or 鋁板**
- 轉軸：**M3 螺絲 + 尼龍墊圈**（當軸銷用）
- 曲柄輪：**3D 列印** 或 **壓克力**

## 尺寸檢查

機器人限制：長 ≤ 400mm、寬 ≤ 400mm

```
估算（以 6 足為例）：

寬度 = 機身寬 + 2 × 連桿展開寬度
     ≈ 100mm + 2 × 75mm = 250mm  ✅ (< 400mm)

長度 = 3 排足的間距 + 前後餘量
     ≈ 2 × 80mm + 100mm = 260mm  ✅ (< 400mm)

高度 = 足抬高 + 機身厚度 + 手臂高度
     ≈ 60mm + 50mm + 手臂  → 不限
```

## SolidWorks 建模建議

1. 先建單一足的 Chebyshev 四桿連桿（2D 草圖）
2. 用「運動研究」驗證足端軌跡是否為近似直線
3. 調整桿長比例直到滿意
4. 複製到 6 足，設定相位差
5. 加入機身底盤、馬達座、手臂座
