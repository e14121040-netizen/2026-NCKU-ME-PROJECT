# App Inventor 遙控 App — 指令對照表

## 藍芽設定
- 模組：HC-05
- 鮑率：38400
- 連線方式：App Inventor 的 `BluetoothClient` 元件

## 指令字元對照

### 步行控制

| 按鈕 | 發送字元 | 功能 | 觸發方式 |
|------|---------|------|----------|
| ▲ | `f` | 前進（全速） | 按住發送 |
| ▼ | `b` | 後退（全速） | 按住發送 |
| ◀ | `l` | 左轉 | 按住發送 |
| ▶ | `r` | 右轉 | 按住發送 |
| ▲(慢) | `F` | 前進（半速） | 按住發送 |
| ▼(慢) | `B` | 後退（半速） | 按住發送 |
| ↺ | `q` | 原地左旋轉 | 按住發送 |
| ↻ | `e` | 原地右旋轉 | 按住發送 |
| ⬜ | `0` | 停止 | **放開時發送** |

### 手臂控制

| 按鈕 | 發送字元 | 功能 |
|------|---------|------|
| 肩↑ | `u` | 肩關節上升 |
| 肩↓ | `d` | 肩關節下降 |
| 肘↑ | `i` | 肘關節上升 |
| 肘↓ | `k` | 肘關節下降 |
| 爪開 | `o` | 夾爪張開 |
| 爪合 | `p` | 夾爪閉合 |
| 歸位 | `h` | 手臂全歸位 |

## App Inventor 設計建議

### 介面佈局（橫式）
```
┌────────────────────────────────────────────┐
│  [連線藍芽]              [斷開]            │
│                                            │
│   步行控制                  手臂控制       │
│       [▲]              [肩↑]  [肘↑]       │
│   [◀] [⬜] [▶]          [歸位]            │
│       [▼]              [肩↓]  [肘↓]       │
│                                            │
│   [↺]     [↻]          [爪開]  [爪合]     │
│                                            │
│   [ 慢速模式 切換 ]                        │
└────────────────────────────────────────────┘
```

### 關鍵 Block 設計

1. **按住/放開控制**（步行按鈕）：
   - `TouchDown` 事件 → 發送移動指令（如 `f`）
   - `TouchUp` 事件 → 發送停止指令（`0`）

2. **單擊控制**（手臂按鈕）：
   - `Click` 事件 → 發送對應字元

3. **藍芽連線**：
   - 使用 `ListPicker` 選擇已配對的 HC-05
   - 連線成功後啟用控制按鈕

## 注意事項

- 學長的 `.aia` 檔案可以直接匯入 App Inventor 修改
- HC-05 只支援 Android（不支援 iOS）
- 建議按鈕要夠大，方便比賽時快速操作
