# 運輸機器人 — 零件清單 (BOM)

> EV3 機器人的零件大多為 LEGO EV3 教具套件，由學校提供借用。  
> 以下列出所需零件與額外消耗品。

| 品項 | 數量 | 說明 |
|------|------|------|
| EV3 Brick 主控 | 1 | 主控電腦 |
| EV3 大馬達 | 2 | 左右輪驅動 (Port B, C) |
| EV3 中馬達 | 1 | 放置機構 (Port A) |
| EV3 顏色感測器 | 3 | 左循跡 (S1) + 右循跡 (S2) + 停止線 (S3) |
| EV3 觸碰感測器 | 1 | 啟動/零件偵測 (S4) |
| EV3 連接線 | ≥7 條 | 馬達 ×3 + 感測器 ×4 |
| LEGO 結構件 | 若干 | 底盤、支架、齒輪 |
| EV3 大輪 | 2 | 直徑 56mm |
| 萬向輪/支撐輪 | 1 | 或用 LEGO 球輪 |
