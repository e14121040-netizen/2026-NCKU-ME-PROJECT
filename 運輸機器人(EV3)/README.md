# 運輸機器人（EV3 全自動循跡）

> 本目錄包含 EV3 運輸機器人的全部資料。

## 系統架構

```
EV3 Brick (Pybricks MicroPython)
    ├── Port D: 左輪大馬達
    ├── Port C: 右輪大馬達
    ├── Port A: 放置機構中馬達
    ├── Port S1: 前下顏色感測器（循跡與標線定位）
    └── Port S4: 觸碰感測器（啟動/零件偵測）
```

## 任務流程

```
等待啟動 → PID 循跡 → 偵測黃色停止線 → 停車放置零件 → 前進跨越黃線
                                                  ↓
                  停車等待下一輪 ← 偵測紅色停止線 ← 繼續循跡
                        ↓
         (觸碰感測器觸發) → 跨越紅線繼續下一輪
```

## 控制方式

- **全自動**（無遙控）
- 單感測器 PID 黑線左緣循跡（持續調參中）
- 循跡感測器同時偵測黃色（放置區）/ 紅色（起點）停止線
- **計畫中**：運輸帶用超音波感測器偵測零件到位，需與取物機器人的承物台/堆高機尺寸配合

## 快速開始

1. 閱讀 [BOM.md](BOM.md) 確認零件
2. 依照 [EV3組裝設計參考.md](機構設計/EV3組裝設計參考.md) 組裝
3. 安裝 [VS Code + EV3 MicroPython 擴充](https://pybricks.com/ev3-micropython/)
4. 將 `transport_robot.py` 部署到 EV3
5. 在場地上調校 PID 參數

## PID 參數調校

程式中的關鍵參數（在 `transport_robot.py` 開頭）：

| 參數 | 預設值 | 說明 |
|------|--------|------|
| `KP` | 0.6 | 比例增益（擺動大→降低，跟線不緊→提高） |
| `KI` | 0.0 | 積分增益（先設0，穩定後再微調） |
| `KD` | 0.3 | 微分增益（抑制震盪） |
| `BASE_SPEED` | 60 mm/s | 基礎行駛速度（速度慢更容易調參） |
| `LINE_THRESHOLD` | 20 | 黑線反射值閾值（可由啟動校準取得） |

### 調校步驟

1. **先只調 KP**（KI=0, KD=0）→ 讓車子能大致跟著線走
2. **加入 KD** → 減少左右晃動
3. **微調 KI** → 消除持續偏移
4. **調整 BASE_SPEED** → 在穩定性和速度間取得平衡
