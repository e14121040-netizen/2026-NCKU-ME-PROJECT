# Theo Jansen Linkage 八足設計參考

## 連桿機構原理

Theo Jansen Linkage 是由荷蘭動態雕塑家 Theo Jansen 發明的 **11 桿連桿機構**，
能將曲柄的旋轉運動轉換為足端的平滑步行軌跡。
與 Chebyshev 四桿機構相比，它的足端軌跡更加仿生，接地段更長且更平直。

```
              固定支點 O
                 ●
                /|\
               / | \
           a  /  |  \ b
             /   |   \
            ● M  |  N ●
           /|    |    |\
       e  / |    |c   | \ d
         /  |    |    |  \
    J ●─/─  |    |    |  \─── K ●
      | f\  |g   |    |h /|
      |   \ |    |    | / |
      |    ●─────●────●  |
      |    L     P     Q  |
      |         / \       |
   i  |        /   \      | j
      |       /     \     |
      ●──────●       ●────●
      R      足端 F        S

  F = 足端 (Foot Point)
```

足端 F 的軌跡：下半段近似水平直線段（接地行走），上半段為橢圓弧（抬腳擺動）。

## Theo Jansen 11 Holy Numbers

這組由 Theo Jansen 透過遺傳演算法 (Genetic Algorithm) 最佳化而得的
**無因次連桿比例**，可等比例縮放至任何尺寸：

```
連桿命名與節點示意：

      O (固定支點)
     / \
    a   b        a = 曲柄 (Crank)
   /     \       b = O→N 連桿
  M       N
 /|       |\
e |       | d    e = M→J, d = N→K
  |g      |h     g = M→L, h = N→Q
  |       |
J ●       ● K
 / f     j \     f = J→L, j = K→S (=Q→S path)
L ●       ● Q
  \       /
 i \     / k     i = L→F (foot), k = Q→F (foot)
    \   /
     \ /
      F (足端)
```

| 連桿代號 | Holy Number (無因次) | 說明 |
|---------|---------------------|------|
| a | 38.0 | 曲柄 (Crank) O→M |
| b | 41.5 | 連桿 O→N |
| c | 39.3 | 固定桿 M→N 間距 (≈ ground) |
| d | 40.1 | N→K |
| e | 55.8 | M→J |
| f | 39.4 | J→L |
| g | 36.7 | M→L |
| h | 65.7 | N→Q |
| i | 49.0 | L→F（足端連桿 1）|
| j | 50.0 | K→S (=Q→S) |
| k | 61.9 | Q→F（足端連桿 2）|
| l | 7.8 | 曲柄偏移 |
| m | 15.0 | 曲柄半徑參考 |

> **註**：不同來源的編號和數值可能略有差異，以上為最常見版本。
> 實際製作時，可在 Python 可視化工具中微調直到軌跡滿意。

## 建議連桿尺寸

### 縮放計算

選定 **縮放因子 (Scale Factor)**，將 Holy Numbers 乘以縮放因子得到實際毫米值。

為符合競賽 **400×400mm** 尺寸限制，建議縮放因子 ≈ **0.7**：

| 連桿代號 | Holy Number | ×0.7 建議尺寸 (mm) |
|---------|-------------|-------------------|
| a (Crank) | 38.0 | **26.6** |
| b | 41.5 | **29.1** |
| c | 39.3 | **27.5** |
| d | 40.1 | **28.1** |
| e | 55.8 | **39.1** |
| f | 39.4 | **27.6** |
| g | 36.7 | **25.7** |
| h | 65.7 | **46.0** |
| i | 49.0 | **34.3** |
| j | 50.0 | **35.0** |
| k | 61.9 | **43.3** |

此尺寸產生約 **~50mm 步幅高度**、**~80mm 步距**。

> **提示**：如果尺寸太小不易加工，可嘗試縮放因子 0.8~1.0，
> 但需重新檢查整機尺寸是否超過 400×400mm。

## 八足配置

```
  俯視圖（機身長軸方向 ↑ = 前進方向）

         前
    足1 ──── 足5
     |        |
    足2 ──── 足6       ← 機身
     |        |
    足3 ──── 足7
     |        |
    足4 ──── 足8
         後

  左側（足1,2,3,4）→ 馬達 A 驅動
  右側（足5,6,7,8）→ 馬達 B 驅動
```

### 交替步態 (Alternating Gait)

同側 4 足的曲柄相位差各 **90°**（360° ÷ 4 = 90°），
左右兩側錯開 **180°**：

```
  左側相位：        右側相位：
  足1 =   0°       足5 = 180°
  足2 =  90°       足6 = 270°
  足3 = 180°       足7 =   0°
  足4 = 270°       足8 =  90°
```

**效果**：任何時刻，同側有約 2~3 足著地，提供穩定支撐。
8 足比 6 足更穩定，步行更平滑。

### 步態週期示意

```
  Phase 0° :  足1,3,7 著地 ─┬─ 穩定支撐
              足2,4,5,6,8   ─┘  部分擺動

  Phase 90°:  足2,4,5,8 著地 ─┬─ 穩定支撐
              足1,3,6,7     ─┘  部分擺動

  Phase 180°: 足3,5,6 著地  ─┬─ 穩定支撐
              足1,2,4,7,8   ─┘  部分擺動

  Phase 270°: 足4,1,7,8 著地 ─┬─ 穩定支撐
              足2,3,5,6     ─┘  部分擺動
```

> 由於 Theo Jansen 足端軌跡的接地段約占 60% 以上，
> 8 足配置下幾乎任何時刻都有 4~5 足著地，非常穩定。

## 材料選擇

| 材料 | 優點 | 缺點 | 適用 |
|------|------|------|------|
| **壓克力 (3mm)** | 便宜、雷切精準、透明美觀 | 易碎（跌落可能斷裂） | 連桿片 |
| **3D 列印 PLA** | 設計自由度高、可做複雜形狀 | 精度略低、層間強度差 | 機身、支架 |
| **鋁板 (2mm)** | 強度高、耐用 | 加工較難、成本高 | 關鍵承力件 |

### 推薦組合
- 連桿片：**壓克力雷切**（便宜快速）
- 機身底盤：**壓克力 or 鋁板**
- 轉軸：**M3 螺絲 + 尼龍墊圈**（當軸銷用）
- 曲柄輪：**3D 列印** 或 **壓克力**

## 尺寸檢查

機器人限制：長 ≤ 400mm、寬 ≤ 400mm

```
估算（以 8 足、縮放因子 0.7 為例）：

寬度 = 機身寬 + 2 × 連桿展開寬度
     ≈ 100mm + 2 × 80mm = 260mm  ✅ (< 400mm)

長度 = 4 排足的間距 + 前後餘量
     ≈ 3 × 60mm + 100mm = 280mm  ✅ (< 400mm)

高度 = 足抬高 + 機身厚度 + 手臂高度
     ≈ 50mm + 50mm + 手臂  → 不限
```

> ⚠ 如果整機尺寸接近 400mm，可以縮小縮放因子或減少排距。
> 務必在 SolidWorks 中 3D 驗證。

## 與 Chebyshev 機構比較

| 特性 | Chebyshev Lambda (舊) | Theo Jansen (新) |
|------|----------------------|------------------|
| 連桿數 | 4 桿 | 11 桿 |
| 足端軌跡 | 近似直線 + 弧線 | 更仿生的橢圓步態 |
| 接地段比例 | ~50% | ~60%+ |
| 機構複雜度 | 低 | 中 |
| 步行穩定性 | 三角步態 6 足 | 交替步態 8 足，更穩 |
| 加工難度 | 較簡單 | 連桿片更多，但幾何單純 |

## SolidWorks 建模建議

1. 先建單一足的 Theo Jansen 11 桿機構（2D 草圖）
2. 用「運動研究」驗證足端軌跡是否為平滑步行路徑
3. 調整縮放因子直到滿意（配合 Python 可視化工具）
4. 複製到 8 足，設定相位差（同側每足差 90°）
5. 加入機身底盤、馬達座、手臂座
6. 檢查總尺寸 ≤ 400×400mm
